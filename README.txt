Версия 0.5 — список всех токенов на сцене

Что заменить в репозитории:
- index.html
- main.js
- styles.css
- heart.svg

После замены:
1. В manifest.json подними version, например до 0.5.0
2. commit + push
3. В Owlbear удали расширение и добавь заново
4. Нажми Ctrl+F5 в комнате

Как работать:
- Открой расширение
- В блоке "Все токены на сцене" нажми "Добавить" у нужного токена
- Токен появится в блоке "Отслеживаемые токены"
- Нажми на него и редактируй значения
